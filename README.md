# MarbyLink

**MarbyLink** 是一款**纯 P2P、无服务器**的家庭局域网通讯软件。  
无需互联网，无需注册账号，只要连上同一个 Wi-Fi 或网线，家人之间就能聊天、传文件、视频通话。所有数据只在您的家庭网络中传输，不经过任何云端服务器。

## 🏠 为什么需要 MarbyLink？

- **绝对去中心化**：没有信令服务器、没有中继、没有云消息队列，全靠局域网广播/组播 + 点对点直连。
- **隐私安全**：所有通信端到端加密（Curve25519 + XSalsa20-Poly1305），数据永不离开您的家庭网络。
- **零配置**：安装即用，自动发现同一局域网内的其他设备，无需输入 IP 或注册账号。
- **完整功能**：聊天、群聊（带密码）、文件传输、音视频通话、面对面快传，满足家庭日常所需。

## ✨ 核心功能（完全基于 PRD 1.0）

### 🔍 设备发现与在线状态
- 软件启动后自动通过 **UDP 广播（端口 8877）** 发现局域网内所有运行本软件的设备。
- 主界面显示在线设备的昵称、设备类型图标（手机/电脑），离线设备显示灰色。
- 每 30 秒发送心跳，超过 90 秒未收到则标记离线。

### 💬 点对点文本聊天
- 从设备列表选择任一在线设备，进入聊天窗口。
- 支持发送纯文本、emoji 表情、图片（缩略图+原图）、短语音（按住录制）。
- 消息显示发送时间和“已送达对方设备”状态（因纯 P2P 无服务器，无“已读”回执）。
- 聊天记录本地 SQLite 存储，支持按关键字搜索。

### 📁 文件/文件夹传输
- 拖拽或点击发送文件/文件夹，支持多文件并发、保留目录结构。
- **断点续传**：传输中断后，下次对方在线可继续传输。
- 实时显示传输进度和速度（MB/s），完成后通知接收方。
- 单个文件最大支持 4GB（兼容 FAT32，实际可更大）。

### 👥 群聊（带密码校验）
- **创建群组**：任意成员可创建群组，生成群 ID（UUID）并设置加入密码（4-8 位数字或字母）。
- **加入群组**：其他成员输入群 ID 和密码，验证通过后加入。
- **群组管理**：创建者可修改密码、踢出成员；成员可主动退出。
- **消息分发**：发送方遍历群内在线成员，分别通过 TCP 连接发送消息副本（无中心服务器）。
- **可选离线队列**：发送方可勾选“等待离线成员上线后自动发送”，消息暂存在本地队列（最多 100 条/成员），成员上线后自动发出。
- 群聊界面显示群名称（可修改）、成员列表（在线/离线），支持文本、图片、文件。

### 📞 音视频通话（无服务器信令）
- 从设备列表或群成员列表中选择一人，发起语音或视频通话。
- **信令机制**：呼叫方通过 UDP 广播发送 `call_request`（含 SDP offer），被叫方收到后回复 `call_response`（含 SDP answer）。每个信令重发 3 次（间隔 200ms），超时则提示无响应。
- 通话建立后使用 **WebRTC** 直连（仅 host 候选，无需 STUN/TURN）。
- 通话界面支持静音、扬声器、挂断、切换前后摄像头（移动端）。

### 📱 面对面快传
- 适用于两人在同一个房间快速传文件，无需查找设备列表。
- **二维码方式**：发送方选择文件生成二维码（包含临时会话 ID、IP、端口），接收方扫码后自动建立 TCP 连接传输。
- **4 位数字码方式**：发送方生成临时 4 位数字，接收方输入该数字，软件自动广播查找发送方并建立连接。
- 临时会话一次性有效，传输完成后自动销毁。

### ⚙️ 设置与个性化
- **个人资料**：修改昵称、头像（从预设图标选择或拍照）。
- **网络**：手动指定 UDP 广播地址（如 `255.255.255.255`）或组播地址（`239.0.0.1`），端口范围可配置。
- **存储**：更改文件保存路径，清理聊天记录。
- **加密**：显示设备公钥指纹，支持验证对方身份（可选高级功能）。
- **外观**：深色/浅色模式，字体大小调整（适合老人）。

## 🔒 安全与隐私

- **端到端加密**：所有文本、文件、音视频流均使用 `libsodium` 的 `crypto_box`（Curve25519 + XSalsa20-Poly1305）。设备首次发现时交换公钥。
- **防伪造**：每条消息附带发送方签名，接收方验证。
- **群聊密钥**：群组拥有独立对称密钥，由创建者生成，并通过点对点加密分发给每个成员（成员输入正确密码后获取）。
- **无外网请求**：软件不会向任何外网 IP 发送数据（包括 DNS 查询），仅限局域网通信。可用 Wireshark 验证。

- **Android/iOS**：扫码下载 APK / IPA（内部分发）
- **Windows/macOS/Linux**：运行安装程序

> 如果您希望在没有互联网的环境下使用，只需将所有设备连接到同一路由器（或交换机）即可。

## 📦 技术架构（简述）

| 模块             | 技术方案                                   |
| ---------------- | ------------------------------------------ |
| 跨平台框架       | Flutter 3.x (Dart)                         |
| 设备发现         | UDP 广播（端口 8877），可选 mDNS            |
| TCP 通信         | `dart:io` Socket，自定义帧格式（长度+数据） |
| 加密             | `sodium_libs` (libsodium Dart 绑定)        |
| 文件传输         | 多线程分块 + 断点续传                       |
| 音视频通话       | `flutter_webrtc` + 自定义广播信令           |
| 本地存储         | `sqflite` (SQLite)                         |
| 二维码/条形码    | `mobile_scanner` + `qr_flutter`            |
